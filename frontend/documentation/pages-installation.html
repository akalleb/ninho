Get Started

  1. StrikingDash
  2. Get Started

Welcome to Next.js! Next.js helps you build modern applications for the web, mobile, or desktop.

For getting started an Next.js application you needs two things as Prerequisites.

Prerequisites

Before you begin, ensure you have the following installed on your system:

  • Node.js - Version 18.0 or higher
  • npm - Version 9.0 or higher (comes with Node.js)
  • Git - For version control (optional)
# Check your Node.js version node --version # Should be v18.0.0 or higher # Check your npm version npm --version # Should be 9.0.0 or higher
Important: Make sure you're using the correct Node.js version. Some packages may not work with older versions.

Installation

Step 1: Extract Files

Extract the template files to your desired location. You can use any folder name you prefer.

Step 2: Install Dependencies

Open your terminal/command prompt, navigate to the template directory, and run:

# Navigate to the template directory cd strikingdash-nextjs # Install all dependencies npm install

This will install all required packages. The process may take a few minutes depending on your internet connection.

Tip: If you encounter any errors during installation, try deleting the node_modules folder and package-lock.json file, then run npm install again.

Step 3: Environment Configuration

Create a .env.local file in the root directory:

# Copy the example file cp .env.example .env.local

Edit .env.local and configure your environment variables (optional):

# Base path (optional - remove for root domain deployment) NEXT_PUBLIC_BASE_PATH=/strikingdash-react # API endpoint (optional - update with your backend API URL) NEXT_PUBLIC_API_ENDPOINT=https://api.example.com
Note: The template uses client-side authentication that works seamlessly with or without a base path. No additional authentication configuration is required.
Tip: If you want to run the app without a base path (on root domain), simply remove or comment out the NEXT_PUBLIC_BASE_PATH line. Authentication will work correctly either way.

Step 4: Run Development Server

Start the development server:

npm run dev

Open your browser and navigate to:

  • With base path: http://localhost:3000/strikingdash-react
  • Without base path: http://localhost:3000
Congratulations! Your template is now running. You should see the login page or dashboard depending on your configuration.
